## 🚀 GETTING STARTED

```bash
# Clone any of our repositories
git clone https://github.com/ZaryxStudios/[PROJECT_NAME].git

# Navigate to project directory
cd [PROJECT_NAME]

# Build with Gradle
./gradlew build

# Run tests
./gradlew test
```
